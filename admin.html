Where this data lives on Vercel

  1. Open Vercel Dashboard, then select the CodeSimplr project.
  2. Go to Storage or Integrations and open the connected Neon Postgres database.
  3. Open Tables and look for `website_signups` for leads and `website_events` for traffic events.
  4. Go to Project Settings, then Environment Variables, and confirm `DATABASE_URL` plus `SIGNUPS_ADMIN_TOKEN` exist for Production.
  5. Go to Analytics in the Vercel project sidebar to enable Vercel Web Analytics for aggregated visitor panels.
